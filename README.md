# steal-react-example

1. Clone this repo.
2. Run `npm install`
3. Start an `http-server` in the root directory.
4. Open in your browser.

You can click on the “Welcome to…” title to see the `onClick` event fire.

This example uses [steal-jsx](https://www.npmjs.com/package/steal-jsx).
